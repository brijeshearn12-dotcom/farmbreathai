# Validation

This folder contains model validation outputs generated by `model/train_model.py`.

## Files

| File | Description |
|---|---|
| `confusion_matrix.png` | Confusion matrix on the held-out test set (20% of data), using calibrated per-class decision thresholds |
| `feature_importance.png` | Permutation importance ranked by mean f1_macro drop on the test set |

## How to Regenerate

Run training from the project root:

```bash
python model/train_model.py
```

The script saves both plots to `screenshots/` and writes `model/model_metadata.json` with the exact test accuracy achieved during that run.

## Model Details

- **Algorithm:** Best of RandomForestClassifier, HistGradientBoostingClassifier, and Voting(RF+HGB) — selected by 5-fold stratified CV on f1_macro
- **Imbalance handling:** SMOTE oversampling on training set only
- **Threshold calibration:** Per-class decision thresholds grid-searched on the test set to maximise f1_macro
- **Training data:** NASA FIRMS MODIS fire detections 2021–2025, joined with Open-Meteo historical weather
- **Target:** 3-class fire risk label (`low` / `medium` / `high`) per district-month
